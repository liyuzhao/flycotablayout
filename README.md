#### Step 1.
Add it in your root build.gradle at the end of repositories:

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

#### Step 2.
Add the dependency

```
	dependencies {
	        implementation 'androidx.appcompat:appcompat:1.0.0'
	        implementation 'com.github.liyuzhao:flycotablayout:v2.0.0'
	}

```
