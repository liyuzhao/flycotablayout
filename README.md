#### Step 1.
Add it in your root build.gradle at the end of repositories:

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

#### Step 2.
Add the dependency

```
	dependencies {
	        implementation 'com.android.support:appcompat-v7:27.0.1'
	        implementation 'com.github.liyuzhao:flycotablayout:v1.0.1'
	}

```
